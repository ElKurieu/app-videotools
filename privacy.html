Privacy Policy

Last updated: March 2026

1. Overview

N8N_AutoPublish respects your privacy. This policy explains what data is accessed and how it is used.

2. Data We Access

  • TikTok account: We access your TikTok account solely to post videos on your behalf using the video.upload scope.
  • Google Drive: We access video files you explicitly select for publishing.

3. Data We Do NOT Collect

We do not collect, store, or sell any personal data. All processing happens locally via n8n automation running on a private server.

4. Third-Party Services

  • TikTok API – for video publishing
  • Google Drive API – for video file access

5. Data Retention

No personal data is retained. Access tokens are stored only in n8n's encrypted credential store on your private server.

6. Contact

For any privacy-related questions, contact the app owner via GitHub.
