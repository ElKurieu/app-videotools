Terms of Service

Last updated: March 2026

1. Description

N8N_AutoPublish is a personal automation tool that posts educational videos from Google Drive to TikTok using the TikTok API.

2. Usage

This application is for personal use only. It interacts with TikTok's Content Posting API to automate video publishing on behalf of the authenticated user.

3. Data

No user data is collected or shared with third parties. The app only accesses TikTok account data necessary for video publishing (video.upload scope).

4. Compliance

This app complies with TikTok's Developer Terms of Service and Platform Guidelines.

5. Contact

For any questions, contact the app owner via GitHub.
